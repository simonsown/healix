# Healix - Trợ lý Sức khỏe Thông minh

Quản lý hồ sơ sức khỏe cá nhân, làm bài test sàng lọc, lưu trữ bệnh án và nhận tư vấn dinh dưỡng từ AI.

## Tính năng

- Bài test sức khỏe (trầm cảm, tiểu đường, ung thư, dinh dưỡng, HPV)
- Phân tích chỉ số máu bằng AI
- OCR đơn thuốc
- Quản trị bộ câu hỏi test

## Công nghệ

- Next.js 16
- TypeScript
- Tailwind CSS v4
- Vercel AI SDK
